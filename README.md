# Lumina Creative

## Getting Started (Local Setup)

To run this project locally:

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/lumina-creative.git
   ```

2. Navigate into the project folder

   ```bash
   cd lumina-creative
   ```

3. Open `index.html` in your browser
   _(or use Live Server in VS Code for a better experience)_

## Responsiveness

The project uses media queries to ensure a consistent and accessible experience across different screen sizes, including optimizations for tablets (`max-width: 768px`) and smaller devices.

---

## Gallery Lightbox

The gallery section integrates **Lightbox2**, allowing users to preview images in a focused modal view without leaving the page. This improves usability and visual presentation.
